# Features

## Overview

The Art Gallery web application provides an interactive and responsive experience for exploring different forms of art. The application is built using HTML, CSS, and JavaScript without relying on external front-end frameworks.

---

## Core Features

### Responsive Navigation

- Fixed navigation bar
- Mobile-friendly navigation menu
- Hamburger menu for smaller screens
- Smooth scrolling between sections

---

### Hero Section

- Full-screen landing section
- Animated heading text
- Call-to-action buttons
- Scroll indicator

---

### Interactive Gallery

The gallery section includes:

- Responsive image grid
- Hover overlay effects
- Artwork titles
- Artwork descriptions
- Smooth entrance animations

---

### Art Types Section

The application presents seven major art forms:

- Painting
- Sculpture
- Literature
- Architecture
- Cinema
- Music
- Theater

Each card uses a CSS flip animation to reveal additional information.

---

### About Section

Provides information about the gallery along with animated statistics.

Displayed statistics include:

- Artists
- Artworks
- Exhibitions

The numbers animate when the section becomes visible.

---

### Contact Section

Users can:

- View contact information
- Enter their name
- Enter their email address
- Send a message using the contact form

The current implementation demonstrates client-side form handling.

---

### Newsletter Subscription

The newsletter section allows users to:

- Enter an email address
- Submit a subscription request

A confirmation message is displayed after submission.

---

### Back to Top Button

After scrolling down the page:

- A floating button appears.
- Clicking the button smoothly scrolls the page back to the top.

---

### Responsive Design

The application adapts to multiple screen sizes including:

- Desktop
- Laptop
- Tablet
- Mobile

---

### Visual Enhancements

The application includes:

- Hover animations
- CSS transitions
- Image overlays
- Animated statistics
- Smooth scrolling
- Card flip animations
