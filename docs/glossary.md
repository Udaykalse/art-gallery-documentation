# Glossary

## Overview

This glossary defines technical terms used throughout the Art Gallery documentation.

---

## Browser

A software application used to access and display web pages, such as Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari.

---

## CSS (Cascading Style Sheets)

A stylesheet language used to control the appearance, layout, colors, spacing, typography, and animations of web pages.

---

## CSS Variables

Reusable values defined in CSS that make it easier to maintain consistent colors, spacing, and styling throughout the application.

---

## DOM (Document Object Model)

A programming interface that represents an HTML document as a tree of objects. JavaScript interacts with the DOM to update page content and respond to user actions.

---

## Event Listener

A JavaScript function that waits for a specific user action, such as a click, scroll, or form submission, and executes code when that event occurs.

---

## HTML (HyperText Markup Language)

The standard markup language used to structure and organize web page content.

---

## JavaScript (JS)

A programming language used to add interactivity and dynamic behavior to web applications.

---

## Responsive Design

A design approach that ensures a website adapts to different screen sizes, including desktops, tablets, and mobile devices.

---

## Semantic HTML

The use of meaningful HTML elements, such as `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`, to improve accessibility and document structure.

---

## Single-Page Application (SPA)

A web application where users interact with multiple sections without loading separate web pages. Although this project does not use a JavaScript framework, it behaves like a single-page application because navigation occurs within a single HTML document.

---

## Smooth Scrolling

A scrolling effect that animates movement between sections of a web page instead of jumping instantly.

---

## Viewport

The visible area of a web page displayed within the browser window. Responsive layouts adjust based on the viewport size.

---

## Version Control

A system for tracking changes to files over time. Git is the version control system used for this project.

---

## Git

A distributed version control system used to manage source code changes.

---

## GitHub

A cloud-based platform for hosting Git repositories, collaborating on projects, and managing documentation.
