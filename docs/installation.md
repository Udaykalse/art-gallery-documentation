# Installation Guide

## Overview

This guide explains how to set up and run the Art Gallery web application on your local machine.

## Prerequisites

Before installing the project, ensure you have the following:

- A modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari)
- Visual Studio Code (recommended)
- Git (optional, for cloning the repository)

## Installation Steps

### Step 1: Clone the Repository

```bash
git clone https://github.com/Udaykalse/art-gallery-documentation.git
```

### Step 2: Navigate to the Project Directory

```bash
cd art-gallery-documentation
```

### Step 3: Open the Project

Open the project folder in your preferred code editor.

Example:

```bash
code .
```

### Step 4: Launch the Application

Open the `index.html` file in your browser.

Alternatively, use the **Live Server** extension in Visual Studio Code:

1. Open the project.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

The application will automatically open in your default browser.

## Verify the Installation

After opening the project, verify that:

- The homepage loads successfully.
- Images are displayed correctly.
- Interactive features respond as expected.
- No errors appear in the browser console.

## Common Installation Issues

| Issue | Solution |
|--------|----------|
| Page does not load | Ensure `index.html` is opened correctly. |
| Styles are missing | Verify that `style.css` is in the project root. |
| JavaScript is not working | Confirm that `script.js` is linked correctly in `index.html`. |
